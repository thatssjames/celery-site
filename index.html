Easy to use

While keeping things simple, Celery Bot is easy to use and install. Even better, this is all free! Whether you like to add bots to your server or like installing them on your profile, Celery Bot's for you!

Support server

While Celery Bot's slash commands can be useful, they can also sometimes be confusing! This is why we have a community of users who can assist if you run into any issues.

Pre-built Functions

Celery Bot comes with tons of pre-built functions to use whenever you have Celery Bot installed. In this case, they are pre-installed for you with slash commands, so you don't need to worry.


Built-in Custom Local Database

Celery Bot provides a powerful custom local database, cb.db. This database allows you to store and retrieve data with ease, making it convenient for managing and persisting Celery Bot information.

  • index.js your main file
  • package.json
  • Directorydatabase your database

Extensions for Added Functionality

Celery Bot - Uncut offers extensions like cb.music to enhance your bot’s capabilities. These extensions provide additional functionalities that you can use to create more interactive and engaging bots.
